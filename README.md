# Design Patterns in Java

This repository contains **practical examples** of the most common **Design Patterns**, implemented in **Java**, based on the explanations and structure from [Refactoring.Guru](https://refactoring.guru/design-patterns).

Each pattern includes:
- A short description of the concept
- Implementation in Java (with clean, easy-to-understand code)
- Example of usage in a `Main` class
